# Описание проекта

Информационная система предназначена для автоматизации учёта:

- электродеталей;
- поставщиков;
- контроллеров;
- спецификаций изделий;
- складских операций;
- сотрудников склада.

Система реализована на ASP.NET Core MVC с использованием Entity Framework Core и SQLite.

# Основные возможности

## Электродетали

Система позволяет:

- добавлять электродетали;
- хранить артикул;
- учитывать остатки на складе;
- хранить поставщика;
- искать детали;
- сортировать таблицу по столбцам.

---

## Поставщики

Функции:

- хранение информации о поставщиках;
- контактные данные;
- поиск;
- сортировка по столбцам.

---

## Контроллеры

Функции:

- хранение готовых изделий;
- учёт цены;
- степень защиты IP;
- складской остаток;
- поиск и сортировка.

---

## Спецификации контроллеров

Спецификация показывает:

какие электродетали входят в состав контроллера.

Для каждого контроллера хранится:

- электродеталь;
- количество деталей на изделие.

---

## Складские операции

Поддерживаются операции:

- поступление;
- сборка;
- продажа.

При выполнении сборки:

- автоматически списываются электродетали;
- остатки контроллеров увеличиваются.

При удалении операции сборки:

- детали возвращаются обратно на склад;
- количество контроллеров уменьшается.

---

## Сотрудники

Система хранит:

- имя;
- фамилию;
- отчество сотрудников.

# Автоматическая логика сборки

## При операции "Сборка"

Система:

1. Находит спецификацию выбранного контроллера.
2. Проверяет остатки деталей.
3. Списывает необходимые компоненты.
4. Увеличивает количество контроллеров на складе.
5. Создаёт запись складской операции.

---

## При удалении операции "Сборка"

Система:

1. Возвращает детали на склад.
2. Уменьшает количество контроллеров.
3. Удаляет запись операции.
